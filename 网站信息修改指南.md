# 网站信息修改指南

所有网站信息集中在一个文件：**`src/config.js`**，改完保存刷新页面即可生效。

---

## 1. 网页标题 / 描述 / 关键词

**文件**：`src/config.js` 第 20~25 行

```javascript
metaData: {
    title: '忘川渡客的个人主页🎉',                // 浏览器标签页标题
    description: '欢迎来到忘川渡客的奇妙世界！',  // 网页描述（搜索结果摘要）
    keywords: 'Leleo,leleo,个人主页,个人网站',    // 网页关键词
    icon: '/favicon.ico'                          // 网站图标（favicon），支持外链
},
```

另外 `index.html` 第 7~18 行也有一份静态的 `<meta>` 标签，建议同步修改，避免刷新瞬间显示旧内容。

## 4. 主题颜色 / 亮度 / 模糊

**文件**：`src/config.js` 第 33~41 行

```javascript
color: {
    themecolor: "#FFFFFF",          // 主题颜色（卡片、按钮的主色调）
    welcometitlecolor: "#FFFFFF",   // 欢迎语标题颜色
    turntablecolor1: "#FFFF00",    // 时钟转盘渐变色一
    turntablecolor2: "#00FFFF"     // 时钟转盘渐变色二
},
brightness: 85,   // 背景亮度，范围 0~100（百分比）
blur: 5,          // 毛玻璃模糊效果，范围 0~20
```

颜色推荐值（十六进制，带 `#`）：

| 风格 | themecolor | welcometitlecolor |
|---|---|---|
| 纯白 | `#FFFFFF` | `#FFFFFF` |
| 薄荷 | `#D1FFEC` | `#7BFFC9` |
| 樱花 | `#FFE0EC` | `#FF9EC2` |
| 深夜蓝 | `#1A2A4A` | `#4A7AAA` |

## 6. 默认壁纸（PC & 移动端）

**文件**：`src/config.js` 第 47~65 行

```javascript
background: {
    "pc": {
        "type": "video",  // "pic" 静态图；"video" 动态视频
        "datainfo": {
            "title": "世界很温柔《龙族》上杉绘梨衣",
            "preview": "预览地址",   // 视频封面或图片缩略图
            "url": "实际壁纸地址"    // 视频 .webm / .mp4 或图片 .jpg / .png / .webp
        }
    },
    "mobile": { ... }  // 移动端同理
},
```

**路径写法**：
- 本地文件：把文件放到 `public/img/wallpaper/...` 下，路径写 `/img/wallpaper/...`
- 网络直链：直接写完整 URL，如 `https://xxx.com/wallpaper.webm`

## 7. 社交按钮

**文件**：`src/config.js` 第 88~101 行

```javascript
socialPlatformIcons: [
    { icon: "mdi-github",    link: "https://github.com/CXF0817" },
    { icon: "mdi-email",     link: "mailto:cwh081714@520.email.cn" },
    { icon: "mdi-qqchat",    link: "https://user.qzone.qq.com/194389314/infocenter" },
    { icon: "mdi-wechat",    link: "/img/你的微信二维码.jpg" },
    { icon: "mdi-play-circle-outline", link: "https://space.bilibili.com/3546604453759103" },
    // { icon: "mdi-music",     link: "https://www.douyin.com/user/YOUR_UID" },
    // { icon: "mdi-domain",    link: "企业微信二维码链接" },
    // { icon: "mdi-twitter",   link: "..." },
    // { icon: "mdi-youtube",   link: "..." },
    // ...
],
```

**图标来源**：[MDI 图标库](https://pictogrammers.com/library/mdi/)，搜索平台名替换 `mdi-xxx` 即可。
不用的行可以加 `//` 注释掉或直接删除。

**各平台 link 格式**：

| 平台 | 格式 | 怎么拿 |
|---|---|---|
| GitHub | `https://github.com/用户名` | 登录后右上角头像菜单 |
| 邮箱 | `mailto:xxx@example.com` | 前缀 mailto: + 邮箱 |
| QQ | `https://user.qzone.qq.com/你的QQ号` | QQ 空间主页 |
| 微信 | 二维码图片路径 `/img/xxx.jpg` 或二维码生成链接 | 把二维码截图放进 `public/img/` |
| B站 | `https://space.bilibili.com/你的UID` | 浏览器打开个人空间，地址栏里就是 |
| 抖音 | `https://www.douyin.com/user/你的UID` | 主页  分享  复制链接 |
| 企业微信 | 二维码链接 | 企业微信管理后台「客户联系」 获取二维码 |

## 9. 背景音乐

**文件**：`src/config.js` 第 112~116 行

```javascript
musicPlayer: {
    server: 'netease',  // 服务提供商
    type:   'playlist', // 歌单类型
    id:     '8904898278' // 歌单 ID
},
```

**参数取值**：

| server | type | 说明 |
|---|---|---|
| `netease` | `playlist` / `artist` / `album` / `search` | 网易云 |
| `tencent` | 同上 | QQ 音乐 |
| `kugou` | 同上 | 酷狗 |
| `bilibili` | 同上 | B站 |

**歌单 ID 获取**：打开歌单页面，地址栏 `?id=` 后面那串数字就是。
例如 `https://music.163.com/#/playlist?id=8904898278`  ID 为 `8904898278`。

## 10. 壁纸列表（设置面板可选）

**文件**：`src/config.js` 第 119~176 行

```javascript
wallpaper: {
    pic:      [ ... ],   // PC 静态壁纸列表
    picMobile:[ ... ],   // 移动端静态壁纸列表
    video:    [ ... ],   // PC 动态壁纸列表
    videoMobile: [ ... ] // 移动端动态壁纸列表
}
```

每项格式：`{ "title": "显示名称", "preview": "缩略图", "url": "实际壁纸" }`

支持本地路径 `/img/...` 或网络直链 `https://...`。
添加新壁纸只需往对应数组里追加一条对象即可。

## 13. 管理员口令

**文件**：`src/config.js` 第 17 行

```javascript
isAdmin: "LELEO_SUPER_SECRET",
```

**进入管理员模式**：浏览器访问一次 `http://你的域名/?admin=你的口令`（只访问一次即可，会记住）。
管理员才能看到设置面板、修改外观/壁纸。普通访客永远看不到。

## 本地文件放哪里

Vite 会把 `public/` 目录下的文件原样输出到网站根目录：

```
public/
 favicon.ico               网站图标
 img/
    avatar.jpg            头像
    微信二维码.jpg
    wallpaper/
        static/           PC 静态壁纸
        static-mobile/    移动端静态壁纸
        dynamic/          PC 动态壁纸（.webm）
        dynamic-mobile/   移动端动态壁纸（.mp4）
 css/                      全局样式（一般不用改）
```

前端引用时路径直接写 `/img/xxx`，Vite 会自动找到。
